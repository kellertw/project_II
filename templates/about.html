The map of our project consists of GeoJson data that we gathered from our database, which can be seen by the markers placed on it. The markers use coordinate location data provided by the U.S. census bureau. Overlaid on the default map is a heat map that shows the frequency of SARs occurring.

We worked between the live server and VS Code to develop the index.html and about.html documents. We used bootstrap and Emmet abbreviation in order to develop the code. We added elements like the favicon and a fixed footer/navbar to make the website more visually interesting.
